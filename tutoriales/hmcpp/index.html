cpp: Hola Mundo

Simplemente es obligatorio empezar los tutoriales de programacion con el famoso “Hola Mundo”. Vamos a colocar el programa linea por linea. Primero necesitamos una forma de decirle al usuario “Hola Mundo”, y para ello usaremos un stream de salida. El stream de salida no viene de gratis con C++, asi que tendremos que incluirlo. Aqui va la primera linea del codigo

#include<iostream>

iostream significa Input Output Stream, como dice el nombre, en este tendremos acceso a los streams de entrada y saida. El stream de entrada lo usaremos en otro momento, de momento usaremos el de salida. El stream de entrada se llama cout, y se encuentra en la libreria estandar. Para acceder a el tendremos que especificar que esta en el espacio de nombres std, que es esto de espacio de nombres? Esa es una buena pregunta, cuya respuesta tendremos despues.

Ya con el stream de salida lo que tenemos que hacer es enviar el texto al mismo, para ello usaremos el operador <<, enviando el texto al stream de esta manera:

std::cout << "Hola mundo";

Aqui hay dos cosas que tenemos que notar, el texto que no sea codigo va entre comillas "" y cada instruccion termina con un punto y coma ;. Se que son muchas cosas a tomar en cuenta, pero te acostumbras con el tiempo te vas a acostumbrar a estos detalles, y con todo lo que puedes hacer con C++ va a valer la pena. Ultimos 2 detalles, voy a cambiar el "Hola mundo" por "Hola mundo\n". Te dejare que investigues que significa el \n, y si tienes ganas de mas, busca ``std::endl. Y el segundo detalle, vamos a poner el codigo en una funcion, una funcion es un bloque de codigo. En nuestros programas de C++ la funcion que va a llevar el codigo principal se va a llamar main` y nos quedaria el programa de esta manera:

#include<iostream>

int main(void) {
    std::cout << "Hola Mundo\n";
    return 0;
}

Notaras en la ultima linea que main regresa un valor de 0 (return 0;). El sistema operativo sabe si los programas corrieron exitosamente si regresan el valor de 0, si regresan cualquier otro valor el programa termino en error.
