Primera Ventana

En el momento en el que abres el proyecto por primera vez vas a tener esto:

    #define PS_UTIL_TIMER
    #define PS_UTIL_LOG

    #include <SDL.h> // En linux <SDL2/SDL.h>
    #include "ps_utility.hpp"
    #include "sDiamond.hpp"


    int main(int argc, char* argv[]) {
        if (!sdmd::init()) {
            sdmd::log_error();
            sdmd::quit();
            return 1;
        }


        sdmd::quit();
        return 0;
    }

Estas lineas definen los macros para que tengamos el timer y el sistema de registros:

    #define PS_UTIL_TIMER
    #define PS_UTIL_LOG

Luego incluimos SDL y sDiamond, como quiero los mensajes voy a incluir tambien ps_utility. Declaramos ps_utility antes de sDiamond.

Luego en el main lo primero que haremos sera iniciar los sitemas. Lo haremos usando la fucnion bool sdmd::init(). Esta da verdadero si se inicia bien o falso en caso de que salga mal, si sale mal leemos el error y cerramos los sistemas con sdmd::quit(). Llamaremos esta funcion siempre antes de cerrar.

Los primero que haremos sera crear una ventana. Entonces crearemos un objeto sdmd::Window, en este caso solo usaremos el constructor default

    sdmd::Window ventana;

Ahora, la creacion de la ventana puede fallar por varios motivos, asi que antes de hacer cualquier cosa con la ventana vamos a revisar que se haya creado correctamente. Para ello usaremos el metodo createdSuccesfully() de la ventana.

    if (!ventana.createdSuccesfully()) {
        sdmd::log_error();
        sdmd::quit();
        return 2;
    }

Y como queremos ver la ventana mas tiempo vamos a poner un delay de 5 segundos antes de que se cierre. Para esto usaremos la funcion sdmd::delay(), esta toma el tiempo en milisegundos.

    sdmd::delay(5000);

Y con esto tenemos nuestra primera ventana.

Y esta esta bien y todo pero nosotros podriamos necesitarla mas alta, o menos ancha, por ende la vamos a construir de la siguiente manera:

    //                  ("Titulo", ancho, alto)
    sdmd::Window ventana("Tutorial", 640, 480);

Y asi tendriamos nuestra primera ventana. El codigo nos queda de la siguiente manera:

    #define PS_UTIL_TIMER
    #define PS_UTIL_LOG

    #include <SDL.h> // En linux <SDL2/SDL.h>
    #include "ps_utility.hpp"
    #include "sDiamond.hpp"


    int main(int argc, char* argv[]) {
        if (!sdmd::init()) {
            sdmd::log_error();
            sdmd::quit();
            return 1;
        }

        sdmd::Window ventana("Tutorial", 640, 480);
        if (!ventana.createdSuccesfully()) {
            sdmd::log_error();
            sdmd::quit();
            return 2;
        }

        sdmd::delay(5000);

        sdmd::quit();
        return 0;
    }
